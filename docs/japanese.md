# BooksNotes - 読書ノートアプリ

## 読書ノートをスマートに管理

BooksNotesは、Amazon Kindleのハイライトを自動でインポートし、読書ノートを効率的に管理できるアプリです。iPhoneとAndroidの両方に対応しています。

## 主な機能
- **自動インポート:** Kindleハイライトを自動でインポート。
